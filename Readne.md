# Exercícios de Lógica com Vetores em Java 🚀

Este repositório contém uma coleção de algoritmos fundamentais para manipulação de arrays (vetores) na linguagem Java. Os exercícios cobrem desde cálculos básicos até algoritmos de ordenação e busca.

## 📋 Lista de Exercícios

Abaixo estão descritos os programas incluídos e suas respectivas funções:

| Arquivo | Descrição |
| :--- | :--- |
| **AnaliseMedia.java** | Calcula a média de um vetor e conta quantos valores estão acima dela. |
| **OrdenaNumeros.java** | Implementação do algoritmo *Bubble Sort* para ordenar números de forma crescente. |
| **BuscaMaiorValor.java** | Identifica o maior número dentro de um vetor e informa sua posição (índice). |
| **IntersecaoDeVetores.java** | Encontra elementos comuns entre dois vetores diferentes. |
| **UniaoDeVetores.java** | Concatena dois vetores de 5 posições em um único vetor de 10 posições. |
| **InverterVetor.java** | Exibe os elementos de um vetor na ordem inversa (de trás para frente). |
| **RemoverElemento.java** | Remove a primeira ocorrência de um número específico de um array. |
| **ContarFrequencia.java** | Conta quantas vezes um determinado número se repete no conjunto. |
| **ContarNumerosPares.java** | Filtra e conta a quantidade de números pares em uma sequência. |
| **SomaElementosVetor.java** | Realiza o somatório total de todos os valores contidos no vetor. |

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** Java
*   **Ferramentas:** Scanner (entrada de dados), Laços de repetição (for/while), Condicionais (if/else).
